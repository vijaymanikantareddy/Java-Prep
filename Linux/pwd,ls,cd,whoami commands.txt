linux commands:

- using pwd command:
	- present working directory 
	Eg: pwd 
	
- using whoami command:
	- display which user currently logged in 
	Eg: whoami 
	
- using cd command:
	- change directory 
	- used to traverse between the folders 
	Eg: cd / 
		cd /bin
		cd /home/kali/downloads 
		cd . - current folder  
		cd .. - parent folder 
		cd ../..
		cd ../../../../../../../../..
		
		
- shortcut to open terminal is ctrl+alt+t 


- using ls command:
	- listing directories 
	- display content of a folder/directory 
	Eg: ls 
		ls -l 
		ls -a 
		ls -al 
		
- using dir command:
	- listing directories 
	- display content of a folder/directory 
	Eg: dir 
		dir -F
