TCP Handshaking:
- used to check whether server is able to communicate or not 
- it is used to establish a connection between server and client 
- 3 way handhskaing 
- client sends hello to the server ( syn packet )
- server sends hello + ack to the client ( syn+ack )
- client sends ack to server ( ack )
